glossary
========

.. glossary::

   feas
      boolean feasibility

   distn
      finite distribution over states, as a python dict

   fset
      set of feasible states, as a python set

   lmap
      a map from feasible state to likelihood, as a python dict

   lhood
      likelihood

   brute
      brute force calculation using enumeration

   nxdistn
      networkx directed graph with probabilities as edge weights

   nxfset
      networkx directed graph with edge presence/absence determining
      set membership

   history
      a map from node to state, as a python dict

   support
      in the jargon of probability and statistics and mathematics,
      the set of points where a function (for example a probability
      density function) is nonzero
