Welcome to nxmctree's documentation!
====================================

.. glossary::

   feas
      boolean feasibility

   distn
      finite distribution over states, as a python dict

   fset
      feasible set of feasible states, as a python set

   lhood
      likelihood

   brute
      brute force calculation using enumeration

   nxdistn
      networkx directed graph with probabilities as edge weights

   nxfset
      networkx directed graph with edge presence/absence determining
      set membership

   history
      a map from node to state, as a python dict
