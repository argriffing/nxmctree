nxmctree.puzzles module¶

nxmctree.puzzles.gen_random_infeasible_systems()[source]¶
nxmctree.puzzles.gen_random_systems(pzero)[source]¶

Sample whole systems for testing likelihood. The pzero parameter indicates sparsity. Yield (T, edge_to_P, root, root_prior_distn, node_to_data_feasible_set).

nxmctree.puzzles.sample_data_feasible_sets(nodes, states, pzero)[source]¶

Return a map from node to feasible state set. states : set of states pzero : probability that any given state is infeasible

nxmctree.puzzles.sample_dict_distn(states, pzero)[source]¶

Return a random state distribution as a dict. Some entries may be zero. states : set of states pzero : probability that any given state has nonzero probability

nxmctree.puzzles.sample_four_node_system(pzero_transition, pzero_other)[source]¶
nxmctree.puzzles.sample_single_node_system(pzero)[source]¶

nxmctree.puzzles.sample_transition_graph(states, pzero)[source]¶

Return a random transition matrix as a networkx digraph. Some entries may be zero. states : set of states pzero : probability that any given transition has nonzero probability
